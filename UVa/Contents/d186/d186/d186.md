## d186: 11461 - Square Numbers
完全平方數就是平方根為整數的整數。例如 1, 4, 81 就是完全平方數。給你兩個整數 a 和 b，請你求出 a 與 b 之間 (含) 有幾個完全平方數。

## C
```C
#include<stdio.h>
#include<math.h>

int main(void)
{
	int a, b;
	
	while(1)
	{
		scanf("%d %d", &a, &b);
		
		if(!a && !b)
			break;
		
		int n1 = sqrt(a);
		int n2 = sqrt(b);
		
		if(n1 * n1 != a)
			n1++;
				
		printf("%d\n", n2 - n1 + 1);
	}
	
	return 0;
} 
```
 * Result : `AC (1ms, 120KB)`
