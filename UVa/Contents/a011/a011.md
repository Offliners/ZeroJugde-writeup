## a011: 00494 - Kindergarten Counting Game
算一算每行有幾個字（word）。

Word的定義是連續的字元（letter: A~Z a~z）所組成的字。

## C
```C
#include<stdio.h>
#include<string.h>
#include<ctype.h>

int main(void)
{
	char str[1000];
	
	while(gets(str) != NULL)
	{
		int count = 0;
		
		for(int i = 0; i < strlen(str); i++)
		{
			if(isalpha(str[i]))
			{
				count++;
				
				while(isalpha(str[i]))
					i++;
			}
		}
		
		printf("%d\n", count);
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 72KB)`

## C++
```c++
#include<iostream>
#include<sstream>

using namespace std;

int main(void)
{
	string s, temp;
	
	while(getline(cin, s))
	{
		int word = 0;
	
		for(int i = 0; i < s.size(); i++)
            if(!isalpha(s[i]))
				s[i] = ' ';
		
		stringstream ss(s);
		while(ss >> temp)
			word++;
		
		cout << word << endl;
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 316KB)`
