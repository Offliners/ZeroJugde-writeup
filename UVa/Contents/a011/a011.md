## a011: 00494 - Kindergarten Counting Game
算一算每行有幾個字（word）。

Word的定義是連續的字元（letter: A~Z a~z）所組成的字。

## C
```C
#include<stdio.h>
#include<string.h>
#include<ctype.h>

int main(void)
{
	char str[1000];
	
	while(gets(str) != NULL)
	{
		int count = 0;
		
		for(int i = 0; i < strlen(str); i++)
		{
			if(isalpha(str[i]))
			{
				count++;
				
				while(isalpha(str[i]))
					i++;
			}
		}
		
		printf("%d\n", count);
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 72KB)`
