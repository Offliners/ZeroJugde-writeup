## d143: 11172 - Relational Operators
有些運算子（operator）是用來檢查兩個數值之間的關係，這種運算子稱為關係運算子（relational operators）。 

給你兩個數值你的工作就是要找出它們之間的關係是(1)第一個大於第二個 (2)第二個小於第一個 (3)兩個一樣大。

## C
```C
#include<stdio.h>

int main(void)
{
	int n;
	long long int a, b;
	
	while(scanf("%d", &n) != EOF)
	{
		for(int i = 0; i < n; i++)
		{
			scanf("%lld %lld", &a, &b);
			
			if(a > b)
				printf(">\n");
			else if(a < b)
				printf("<\n");
			else
				printf("=\n");
		}
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 64KB)`
