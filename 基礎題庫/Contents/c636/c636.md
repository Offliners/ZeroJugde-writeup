## c636: 十二生肖
民國元年 ( 1912 ) 是鼠年。

給定若干個民國年份   -100 ~ 107

請輸出該年生肖為何。

## C
```C
#include<stdio.h>

char symbol[12][5] = {"豬", "鼠", "牛", "虎", "兔", "龍", "蛇", "馬", "羊", "猴", "雞", "狗"};

int main(void)
{
	int n;
	
	while(scanf("%d", &n) != EOF)
		printf("%s\n", symbol[(n < 0 ? n + 121 : n) % 12]);
	
	return 0;
} 
```
