## c418: Bert的三角形 (1)
Bert 想要一個 n 層的三角形，第 i 層就要有 i 個 " * "

請你寫個程式幫幫可憐的 Bert ~~

## C
```C
#include<stdio.h>

int main(void)
{
	int n;
	
	while(scanf("%d", &n) != EOF)
	{
		for(int i = 0; i < n; i++)
		{
			for(int j = 0; j <= i; j++)
				printf("*");
			printf("\n");
		}
	}
	
	return 0;
} 
```
 * Result : `AC (2ms, 100KB)`
