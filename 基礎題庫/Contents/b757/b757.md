## b757: 頸美椰子樹
PA 頸美椰子樹
頸美女中是一所追求漂亮脖子的好學校，裏頭的夯夯是一個擁有傲人身材的高二生。

「景美椰子樹，是夯夯的大腿，是夯夯的手臂」

大腿、手臂，雖然都很粗，但在GOOGLE地圖上還是能發現其中明顯的差異。
最近夯夯也到了想減肥的花樣年紀了，
假設，大腿半徑瘦下來 R，手臂半徑瘦下來 r，
我們可以立出標準式: R = (r*9)/5 + 32
很巧的，這也是華氏跟攝氏的轉換公式，
剛剛我們正量完夯夯的手臂瘦下了 r 公尺
由於大腿實在太粗了不好測量，想請學弟用程式幫我們算算他的大腿瘦了多少

## C
```C
#include<stdio.h>

int main(void)
{
	double r, R;
	
	while(scanf("%lf", &r) != EOF)
		printf("%g\n", r * 9 / 5 + 32);
		
	return 0;
}
```
 * Result : `AC (2ms, 104KB)`
