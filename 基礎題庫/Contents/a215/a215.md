## a215: 迴文
明明是一個愛數（ㄕㄨˇ）數（ㄕㄨˋ）的好學生，這天媽媽叫他從 n 開始數，下一個數字是 n+1，再下一個數字是 n+2，以此類推。

媽媽想知道，明明數了幾個數字之後，他數過的這些數字的總和會超過 m。請幫助明明的媽媽吧。

## C
```C
#include<stdio.h>

int main(void)
{
	int n, m;
	
	while(scanf("%d %d", &n ,&m) != EOF)
	{
		int sum = 0;
		int i = 0;
		 
		while(sum <= m)
		{
			sum += n + i;
			i++;
		}
		
		if(!i)
			i++;
			
		printf("%d\n", i);
	}
	
	return 0;
}
```
 * Result : `AC (1ms, 96KB)`
