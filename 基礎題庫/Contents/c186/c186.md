## c186: 蝸牛老師的點名單
蝸牛老師新開了一門課，第一堂課時拿了一張白紙讓學生簽到。現在他需要用簽到單上的名字製作一份點名單。

## C
```C
#include<stdio.h>
#include<string.h>

int main(void)
{
	char names[1001];
	char del[2] = " ";
	char *token;
	
	while(gets(names) != NULL)
	{
		token = strtok(names, del);
		
		while(token != NULL)
		{
			printf("%s\n", token);
			token = strtok(NULL, del);
		}
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 84KB)`
