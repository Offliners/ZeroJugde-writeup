## a015: 矩陣的翻轉
已知一(m x n)矩陣A，我們常常需要用到另一個將A中之行與列調換的矩陣。這個動作叫做矩陣的翻轉。

## C
```C
#include<stdio.h>

int main(void)
{
	int row, col;
	int arr[100][100];
	int arrT[100][100];
	
	while(scanf("%d %d", &row, &col) != EOF)
	{
		for(int i = 0; i < row; i++)
			for(int j = 0; j < col; j++)
				scanf("%d", &arr[i][j]);
		
		for(int i = 0; i < col; i++)
		{
			for(int j = 0; j < row; j++)
			{
				arrT[i][j] = arr[j][i];
				printf("%d ", arrT[i][j]);
			}
			printf("\n");
		}
			
	}
	
	return 0;
}
```
 * Result : `AC (1ms, 88KB)`
