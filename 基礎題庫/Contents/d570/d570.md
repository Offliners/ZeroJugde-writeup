## d570: 神龍見首不見尾
給你一個正整數，請逐次去掉尾數並顯示出來。

## C
```C
#include<stdio.h>

int main(void)
{
	int n;
	
	while(scanf("%d", &n) != EOF)
	{
		while(n > 0)
		{
			printf("%d\n", n);
			n /= 10;
		}
	}
	
	return 0;
} 
```
 * Result : `AC (1ms, 100KB)`

## C++
```C++
#include<iostream>

using namespace std;

int main(void)
{
	int n;
	
	while(cin >> n)
	{
		while(n > 0)
		{
			cout << n << endl;
			n /= 10;
		}
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 328KB)`
