## b970: 我不說髒話 (續)
文文上次說髒話被老師罰在黑板上寫 50 遍「I don't say swear words!」，結果他只寫了 45 遍就跑出去玩了，以為老師不會發現。這次老師要求他罰寫的每一遍都要加上編號。

## C
```C
#include<stdio.h>

int main(void)
{
	int n;
	while(scanf("%d", &n) != EOF)
		for(int i = 0; i < n; i++)
			printf("%d. I don't say swear words!\n", i + 1);
			
	return 0;
}
```
