## a044: 空間切割
對任意正整數n，空間中的n 個平面最多可將空間切成幾個區域？

## C
```C
#include<stdio.h>

int slice(int);

int main(void)
{
	int num;
	
	while(scanf("%d", &num) != EOF)
		printf("%d\n", slice(num));
		
	return 0;
}

int slice(int n)
{
	return n == 1 ? 2 : slice(n-1) + ((n * n - n + 2) / 2);
}
```
 * Result : `AC (17ms, 92KB)`
