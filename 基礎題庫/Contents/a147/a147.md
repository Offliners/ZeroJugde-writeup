## a147: Print it all
大於 0、整數、不可以被 7 整除、小於 n，請輸出所有可能的數字。

## C
```C
#include<stdio.h>

int main(void)
{
	int n;
	
	while(scanf("%d", &n) != EOF)
	{
		for(int i = 1; i < n; i++)
		{
			if(i % 7 == 0)
				continue;
			printf("%d ", i);
		}
		printf("\n");
	}
	
	return 0;
}
```
 * Result : `AC (4ms, 56KB)`
