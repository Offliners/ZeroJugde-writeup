## a010: 因數分解
各位在國小時都學過因數分解，都瞭解怎麼樣用紙筆計算出結果，現在由你來敎電腦做因數分解。

因數分解就是把一個數字，切分為數個質數的乘積，如 12=2^2 * 3

其中, 次方的符號以 ^ 來表示

## C
```C
#include<stdio.h>

int main(void)
{
  int num;
  int counter = 0;

  while(scanf("%d", &num) != EOF)
  {
    for(int i = 2; i < 1000000; i++)
    {
      if(num == 1)
        break;
      else
      {
        while(1)
        {
          if(num % i == 0)
          {
            num /= i;
            counter++;
          }
          else
          {
            if(counter != 0)
            {
              printf("%d", i);
              if(counter > 1)
                printf("^%d", counter);
              if(num > 1)
                printf(" * ");
            }
            counter = 0;
            break;
          }
        }
      }
    }
    printf("\n");
  }

  return 0;
}
```
