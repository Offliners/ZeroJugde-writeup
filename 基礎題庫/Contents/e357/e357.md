## e357: 遞迴函數練習
定義一個函數F(x)，

若x = 1, 則 F(x) = 1

若x 為偶數，則 F(x) = F(x/2)

其餘狀況，F(x) = F(x - 1) + F(x + 1)

## C
```C
#include<stdio.h>

long int F(long int);

int main(void)
{
	long int x;
	
	while(scanf("%ld", &x) != EOF)
		printf("%ld\n", F(x));
}

long int F(long int x)
{
	if(x == 1)
		return 1;
	else if(!(x % 2))
		return F(x / 2);
	else
		return F(x - 1) + F(x + 1);
}
```
