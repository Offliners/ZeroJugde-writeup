## a104: 迴文
迴文的定義為正向，反向讀到的字串均相同

如：abba , abcba ... 等就是迴文

 請判斷一個字串是否是一個迴文？

## C
```C
#include<stdio.h>
#include<stdlib.h>

int cmp(const void *, const void *);

int main(void)
{
	int num;
	int data[1000];
	
	while(scanf("%d", &num) != EOF)
	{
		int i;
		for(i = 0; i < num; i++)
			scanf("%d", &data[i]);
			
		qsort(data, num, sizeof(int), cmp);
		
		for(i = 0; i < num; i++)
			printf("%d ", data[i]);
		printf("\n");
	}
	
	return 0;
}

int cmp(const void *a, const void *b)
{
    return *(const int*)a - *(const int*)b;
}
```
 * Result : `AC (3ms, 72KB)`
