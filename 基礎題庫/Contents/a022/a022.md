## a022: 迴文
迴文的定義為正向，反向讀到的字串均相同

如：abba , abcba ... 等就是迴文

 請判斷一個字串是否是一個迴文？

## C
```C
#include<stdio.h>
#include<string.h>

int main(void)
{
	char str[1000];
	int flag;
	
	while(scanf("%s", str) != EOF)
	{
		flag = 0;
		
		for(int i = 0, j = strlen(str) - 1; i < j; i++, j--)
		{
			if(str[i] != str[j])
			{
				flag = 1;
				break;
			}
		}
		
		if(flag)
			printf("no\n");
		else
			printf("yes\n");
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 88KB)`
