## a148: You Cannot Pass?!
你考了 n 科筆試題目，每科的滿分都是 100 分。老師說，如果平均大於 59 你就過關了。

## C
```C
#include<stdio.h>

int main(void)
{
	int n;
	double score, avg;
	
	while(scanf("%d", &n) != EOF)
	{
		double sum = 0;
		int i = 0;
		for(; i < n; i++)
		{
			scanf("%lf", &score);
			sum += score;
		}
			
		avg = sum / n;
		
		if(avg > 59)
			printf("no\n");
		else
			printf("yes\n");
	}
	
	return 0;
}
```
 * Result : `AC (1ms, 76KB)`
