## a005: Eva 的回家作業
Eva的家庭作業裏有很多數列填空練習。

填空練習的要求是：

已知數列的前四項，填出第五項。因 為已經知道這些數列只可能是等差或等比數列，她決定寫一個程式來完成這些練習。

## Solution
```C
#include<stdio.h>

int main(void)
{
  int num[4];
  int t;
  scanf("%d",&t);
  if((t>=0)&&(t<=20))
  {
      for(int j = 0;j<t;j++)
      {
        for(int i = 0;i<4;i++)
            scanf("%d",&num[i]);
        if(num[3]-num[2]==num[2]-num[1])
            printf("%d %d %d %d %d\n",num[0],num[1],num[2],num[3],num[3]+(num[2]-num[1]));
        else if(num[3]/num[2]==num[2]/num[1])
            printf("%d %d %d %d %d\n",num[0],num[1],num[2],num[3],num[3]*(num[2]/num[1]));
        else
            break;
      }
  }
  return 0;
}
```
