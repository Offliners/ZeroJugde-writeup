## b367: 翻轉世界
有位勇士排除萬難後,來到了最後一關-----[翻轉世界]

他發現所有物品都翻轉了180度

如果要繼續前進必須做出一項選擇,也就是找到翻轉後不會改變的東西!

 例如:

110

000

011 

若翻轉180度:

110

000

011 

現在給你一張圖 

請你幫忙這位勇士吧,因為除了必須判斷這個問題以外,還必須打倒翻轉世界的怪物才行

所以請你寫一個程式幫他 

## C
```C
#include<stdio.h>

int main(void)
{
	int T, N, M;
	int mat[122];
	
	while(scanf("%d", &T) != EOF)
	{
		for(int i = 0; i < T; i++)
		{
			scanf("%d %d", &N, &M);
			
			for(int j = 0; j < N * M; j++)
				scanf("%d", &mat[j]);
			
			int head = 0;
			int tail = M * N - 1;
			while(mat[head] == mat[tail])
			{
				if(head > tail)
					break;
				
				head++;
				tail--;
			}
			
			printf("%s\n", head < tail ? "keep defending" : "go forward");
		}
	}
	
	return 0;
}
```
