## d050: 妳那裡現在幾點了？
珊珊到了美國猶他州的楊百翰大學之後，文文禁不住對她的思念，常常想打電話給她，卻又擔心在美國的她是不是在睡覺。

好不容易鼓起勇氣打通了電話，第一句就先問：「妳那裡現在幾點了？」請你幫文文寫一個程式，輸入台灣時間後，算出美國山區時間。

## C
```C
#include<stdio.h>

int main(void)
{
	int h;
	
	while(scanf("%d", &h) != EOF)
		printf("%d\n", (h + 9) % 24);
		
	return 0;
}
```
 * Result : `AC (2ms, 84KB)`
