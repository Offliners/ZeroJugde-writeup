## e976: Will You Make it?
什麼樣的惡夢！ 您起床很晚，仍然必須去CodeWars。 請問你有空嗎？

## C
```C
#include<stdio.h>

int main(void)
{
	int h, m, s;
	
	while(scanf("%d %d %d", &h, &m, &s) != EOF)
	{
		if(h * s >= m)
			printf("%d %d %d. I will make it!\n", h, m, s);
		else
			printf("%d %d %d. I will be late!\n", h, m, s);
	}
	
	return 0;
}
```
