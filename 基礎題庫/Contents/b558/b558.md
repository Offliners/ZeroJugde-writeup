## b558: 求數列第 n 項
有一個數列，第一項的值為 1，第二項的值為第一項加 1，第三項的值為第二項加 2，第四項的值為第三項加 3 ... 第 k 項的值為第 k-1 項的值加上 k-1。

給一個數字 n，請印出這個數列的第 n 項。

## C
```C
#include<stdio.h>

int main(void)
{
	int n;

	while(scanf("%d", &n) != EOF)
		printf("%d\n", 1 + n * (n - 1) / 2);
		
	return 0;
}
```
 * Result : `AC (1ms, 84KB)`
