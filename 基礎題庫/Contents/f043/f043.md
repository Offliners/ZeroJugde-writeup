## f043: 1. 小豪的回家作業 (Homework)

## C
```C
#include<stdio.h>

int main(void)
{
	long int R, A, B;
	
	while(scanf("%ld %ld", &R, &A) != EOF)
	{
		if(R == A)
			A -= 3;
		
		B = R - A;
		if(A < B)
			printf("%ld+%ld=%ld\n", A, B, R);
		else
			printf("%ld+%ld=%ld\n", B, A, R);
	}
	
	return 0;
}
```
 * Result : `AC (7ms, 108KB)`
