## c760: 蝸牛老師的點名單 (再續)
蝸牛老師新開了一門課，第一堂課時拿了一張白紙讓學生簽到。現在他需要用簽到單上的名字製作一份點名單。當他開始這個工作時，他發現現在的學生實在太偷懶，連寫名字時字首都沒大寫。

## C
```C
#include<stdio.h>

int main(void)
{
	char names[1000];
	
	while(scanf("%s", names) != EOF)
	{
		names[0] -= 32;
		puts(names);
	}
	
	return 0;
} 
```
 * Result : `AC (2ms, 88KB)`

## C++
```C++
#include<iostream>

using namespace std;

int main(void)
{
	int num;
	
	while(cin >> num)
		cout << num * num - num + 2 << endl;
		
	return 0;
}
```
 * Result : `AC (11ms, 304KB)`

## Python
```python
import sys

for num in sys.stdin:
    num = int(num)
    print(num ** 2 - num + 2)
```
 * Result : `AC (23ms, 3.8MB)`
