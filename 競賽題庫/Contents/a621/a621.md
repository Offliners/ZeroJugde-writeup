## a621: Powers of Two
讀入一個非負整數 N (範圍 0 到 20) 並印出 2 的次方從 0 到 n。你不需要驗證整數是否在範圍內。

## C
```C
#include<stdio.h>
#include<math.h>

int main(void)
{
	int n;
	
	while(scanf("%d", &n) != EOF)
		for(int i = 0; i <= n; i++)
			printf("2^%d = %.0lf\n", i, pow(2, i));
			
	return 0;
}
```
 * Result : `AC (1ms, 136KB)`
