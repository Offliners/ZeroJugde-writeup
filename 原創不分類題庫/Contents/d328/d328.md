## d328: S△=？
通过△ABC内部一点Q引平行于三角形三边的直线，这些直线分三角形为六个部分，已知三个平行四边形部分的面积为S1，S2，S3，求△ABC的面积。

## C
```C
#include<stdio.h>

int main(void)
{
	long int s1, s2, s3;
	double area;
	
	while(scanf("%ld %ld %ld", &s1, &s2, &s3) != EOF)
	{
		area = s1 + s2 + s3 + (s2 * s3 / 2.0 / s1) + (s1 * s3 / 2.0 / s2) + (s1 * s2 / 2.0 / s3);
	
		printf("%.2lf\n", area);
	}
	
	return 0;
}
```
 * Result : `AC (2ms, 80KB)`
