## d786: 三、平均值
已知一n個元素的整數數列，計算出這組數列的平均值，四捨五入至小數第2位。

## C
```C
#include<stdio.h>

int main(void)
{
	int n, t, num;
	
	while(scanf("%d", &n) != EOF)
	{	
		for(int i = 0; i < n; i++)
		{
			int sum = 0;
			
			scanf("%d", &t);
			for(int j = 0; j < t; j++)
			{
				scanf("%d", &num);
				sum += num; 
			}
			
			printf("%.2lf\n", (double)sum / t);		
		}
	}
	
	return 0;
}
```
 * Result : `AC (1ms, 76KB)`
