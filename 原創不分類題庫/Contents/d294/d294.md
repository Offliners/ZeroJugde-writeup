## d294: 算算算....Easy
各位是高三一定會算矩形內有幾個矩形，吧？！

本題很簡單，指輸入兩個數 n m ，即矩形長寬。

只要輸出有幾個矩形。

例如有一個長二十寬十五，很直覺的想到C二十一取二乘以C十六取二，答案似乎是......

## C
```C
#include<stdio.h>

int main(void)
{
	int n, m;
	
	while(scanf("%d %d", &n, &m) != EOF)
		printf("%d\n", n * (n + 1) * m * (m + 1) / 4);
		
	return 0;
} 
```
 * Result : `AC (14ms, 84KB)`
