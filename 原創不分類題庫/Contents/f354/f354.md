## f354: 好多好多 ~ 好多正方形
(參考 e895. 好多正方形 、 e896. 好多好多正方形 )

在一條長為 L 的通道上堆放正方形的貨品，正方形的邊長不可超過 L，且邊長皆為整數(最小邊長為 1)，

貨品也不能相互堆疊。每次都必須把長度 L 的整個通道堆滿。

這次倉庫管理員希望能將貨品堆成左右對稱的型態，試問有多少種的堆放方式？

## C
```C
#include<stdio.h>

void tableInit(void);

int table[1000001];

int main(void)
{
	int L;
	
	tableInit();
	while(scanf("%d", &L) != EOF)
		printf("%d\n", table[L]);
	
	return 0;
}

void tableInit()
{	
	table[0] = 1;
	
	for(int i = 1; i < 1000001; i++)
	{
		table[i] = table[i - 1];
		if(!(i & 1))
		{
			table[i] *= 2;
			
			if(table[i] >= 10007)
				table[i] -= 10007;
		}
	}
}
```
 * Result : `AC (8ms, 3.9MB)`
