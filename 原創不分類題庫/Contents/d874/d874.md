## d874: 4. 階梯問題
有一段距離 L ，

而你每次走的距離只能是前一次的±1，或者是不變

而一開始你只能走 1 ，到達終點的時候，你要恰好走 1 到達

例如: L = 10

1 2 3 2 1 1

例如: L = 9

1 2 3 2 1

請你輸出最少步數，走法可能有很多，要出答案可能太多，因此不理會

## C
```C
#include<stdio.h>
#include<math.h>

int main(void)
{
	int L;
	
	while(scanf("%d", &L) != EOF)
	{
		int temp = (int)sqrt(L);
		int step = 2 * temp - 1;
		
		step += (L - temp * temp) / temp;
		
		if(L % temp)
			step++;
		
		printf("%d\n", step);
	}
	
	return 0;
}
```
 * Result : `AC (7ms, 116KB)`
