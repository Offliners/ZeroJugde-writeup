## C
```C
#include<stdio.h>

int main(void)
{
	long long int n;
	
	while(scanf("%lld", &n) != EOF)
		for(long long int i = 0; i < n; i++)
			printf("=_=|||\n");
	
	return 0;
}
```
 * Result : `AC (2ms, 76KB)`

## C++
```C++
#include<iostream>
```
