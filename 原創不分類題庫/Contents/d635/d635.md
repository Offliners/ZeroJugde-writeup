## d635: 幸運777？luck
你說什麼？777=511？怎麼可能…？

因為這裡是八進位的777轉換成10進位的511了～(好爛ˇ)

給你十進位的數字，請你把他們轉成八進位好嗎？

## C
```C
#include<stdio.h>

int main(void)
{
	int n;
	
	while(scanf("%d", &n) != EOF)
	{
		if(n < 0)
			printf("-1\n");
		else
			printf("%o\n", n);
	}
	
	return 0;
}
```
