## c730: 一天一那吾
那吾克熱家的院子裡有一棵蘋果樹，每到秋天，樹上就會結出10個蘋果。蘋果成熟的時候，那吾克熱就會跑去摘蘋果。

那吾克熱有個30公分高的板凳，當他不能直接用手摘到蘋果的時候，會踩到板凳上再試試。

現在已知10個蘋果到地面的高度，以及那吾克熱把手伸直的時候能夠達到的最大高度，請幫那吾克熱算一下他能夠摘到的蘋果的數目。

## C
```C
#include<stdio.h>

int main(void)
{
	int apple[10];
	int hand;
	
	for(int i = 0; i < 10; i++)
	if(scanf("%d", &apple[i]) == EOF)
		break;

	scanf("%d", &hand);

	int count = 0;
	for(int i = 0; i < 10; i++)
		if(apple[i] <= hand + 30)
			count++;

	printf("%d\n", count);
	
	return 0;
}
```
 * Result : `AC (1ms, 96KB)`
