## d493: 入门求幂题(求幂系列题1)
求出a^n的值。

## C
```C
#include<stdio.h>
#include<math.h>

int main(void)
{
	long int a, n;
	
	while(scanf("%ld %ld", &a, &n) != EOF)
		printf("%.0lf\n", pow((double)a, n));
	
	return 0;
} 
```
 * Result : `AC (2ms, 148KB)`

## C++
```C++
#include<iostream>
#include<cmath>
#include<iomanip>

using namespace std;

int main(void)
{
	long int a, n;
	
	while(cin >> a >> n)
		cout << fixed << setprecision(0) << pow(a, n) << endl;
	
	return 0;
}
```
 * Result : `AC (2ms, 336KB)`
