## d493: 入门求幂题(求幂系列题1)
求出a^n的值。

## C
```C
#include<stdio.h>
#include<math.h>

int main(void)
{
	long int a, n;
	
	while(scanf("%ld %ld", &a, &n) != EOF)
		printf("%.0lf\n", pow((double)a, n));
	
	return 0;
} 
```
