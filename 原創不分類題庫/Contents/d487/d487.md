## d487: Order's computation process
1751年，歐拉以大寫字母 M 表示 m 階乘

 M=1．2．3．...．m 　

 1799 年，魯非尼在他出版的方程論著 述中，則以小寫字母 π 表示 m 階乘。

 而在 1813年，高斯則以 Π(n) 來表示 n 階乘。

 用來表示 n 階乘的方法起源於英國，但仍未能確定始創人是誰。

 直至1827年，由於雅萊特的建議而得到流行，現在有時也會以這個符號作為階乘符號。

 而最先提出階乘符號 n! 的人是克拉姆（1808），後來經過歐姆等人的提倡而流行，直至現在仍然通用。

## C
```C
#include<stdio.h>

int fac(int);

int main(void)
{
	int n, temp;
	
	while(scanf("%d", &n) != EOF)
	{
		printf("%d! = ", n);
		temp = fac(n);
		printf("%d\n", temp);
	}
	
	return 0;
}

int fac(int num)
{
	if(!num || num == 1)
	{
		printf("1 = ");
		return 1;
	}
	
	printf("%d * ", num);
	return fac(num - 1) * num;
} 
```
 * Result : `AC (1ms, 72KB)`

## C++
```C++
#include<iostream>

using namespace std;

int fac(int);

int main(void)
{
	int n, temp;
	
	while(cin >> n)
	{
		cout << n << "! = ";
		temp = fac(n);
		cout << temp << endl;
	}
	
	return 0;
}

int fac(int num)
{
	if(!num || num == 1)
	{
		cout << "1 = ";
		return 1;
	}
	
	cout << num << " * ";
	return fac(num - 1) * num;
} 
```
 * Result : `AC (2ms, 320KB)`
