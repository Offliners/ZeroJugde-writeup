## d579: 兩條線
「真是三條線點點點｜｜｜。。。」

生活中我們常常會發出這樣的感慨，同樣的，在實數的世界中，他們也會發出「兩條線」的感慨，此時它的值也會發生奇妙的改變，如下所示；

|105| = 105

|-270| = 270

|-0.752| = 0.752

之類的。總之，現在請你寫一個程式可以計算某個實數加上「兩條線」後的值。

## C
```C
#include<stdio.h>
#include<math.h>

int main(void)
{
	double n;
	
	while(scanf("%lf", &n) != EOF)
		printf("|%.04lf|=%.04lf\n", n, fabs(n));
	
	return 0;
}
```
 * Result : `AC (2ms, 80KB)`
