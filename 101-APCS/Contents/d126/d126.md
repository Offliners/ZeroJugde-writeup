## d126: 一、牧场围栏
Z先生是一个大牧场主，他饲养的奶牛每年都源源不断地运往世界各地，我们的故事也由此产生了……

 

Z先生有一个矩形牧场，它的长和宽分别用 a 和 b （单位：米）表示，现在需要再牧场四周修建围栏，请编一个程序计算牧场的周长以确定用料的多少。

## C
```C
#include<stdio.h>

int main(void)
{
	int a, b;
	
	while(scanf("%d %d", &a, &b) != EOF)
		printf("%d\n", 2 * (a + b));
		
	return 0;
}```
 * Result : `AC (1ms, 84KB)`
